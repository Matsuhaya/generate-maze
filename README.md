# [迷路作成プログラム](http://grayhorse5.sakura.ne.jp/generate-maze/)

## Description

本プログラムは、自動で迷路を作成する。使用言語は JavaScript。迷路の生成は壁伸ばし法を、正解ルートの探索には幅優先探索をそれぞれ用いた。
